Column layout

Example

By default columns will be set like this:

  • s: [1]
  • m: [1, 1]
  • l: [1, 1, 1]
import ColumnLayout from "@mittwald/flow-react-components/ColumnLayout";
import TextField from "@mittwald/flow-react-components/TextField";
import Label from "@mittwald/flow-react-components/Label";

<ColumnLayout>
  <TextField>
    <Label>First name</Label>
  </TextField>
  <TextField>
    <Label>Last name</Label>
  </TextField>
  <TextField>
    <Label>Street</Label>
  </TextField>
  <TextField>
    <Label>House number</Label>
  </TextField>
</ColumnLayout>;

Custom values

Larger sizes will inherit from smaller sizes, if they are not set:

  • s: [1] (default)
  • m: [2, 1]
  • l: [2, 1] (inherited)
import ColumnLayout from "@mittwald/flow-react-components/ColumnLayout";
import TextField from "@mittwald/flow-react-components/TextField";
import Label from "@mittwald/flow-react-components/Label";

<ColumnLayout m={[2, 1]}>
  <TextField>
    <Label>Street</Label>
  </TextField>
  <TextField>
    <Label>House number</Label>
  </TextField>
</ColumnLayout>;
